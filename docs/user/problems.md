# 📝 题目系统

## 工作流

```
投稿 ──→ 待审核 ──→ 批准 ──→ 发布
                  └──→ 拒绝
```

| 状态 | 说明 | 可见性 |
|------|------|--------|
| `pending` | 已投稿，等待审核 | 仅作者和管理员 |
| `approved` | 审核通过，可发布 | 有权限的用户 |
| `rejected` | 审核未通过 | 仅作者和管理员 |
| `published` | 已发布 | 公开或按 ACL |

## 投稿题目

需要 `submit_problem` 权限。

1. 进入「题目」页面
2. 点击「投稿」
3. 填写题目信息：
   - **标题**
   - **难度**（从预设等级中选择）
   - **内容**（Markdown 格式，支持 LaTeX 公式、代码高亮）
   - **题解**（可选，Markdown）
   - **外部链接**（可选）
   - **备注**（可选，仅管理员可见）
4. 提交后状态为 `pending`

### Markdown 支持

- 标准 Markdown
- LaTeX 数学公式（`$...$` 行内，`$$...$$` 块级）
- 代码高亮（\`\`\`language）
- Luogu 风格 Callout：

```
> [!NOTE]
> 这是一个提示

> [!WARNING]
> 这是一个警告

> [!TIP]
> 这是一个建议
```

## 审核题目

需要 `approve_problem` 权限。

1. 进入题目列表 → 点击「待审核」标签
2. 查看投稿详情
3. 操作：
   - **批准**：设为 `approved`，可后续发布
   - **拒绝**：设为 `rejected`，作者可查看拒绝状态
4. 审核操作会自动通知投稿作者

## 认领与验题

需要 `approve_problem` 权限。

**认领题目**：
1. 在题目详情页点击「认领」
2. 成为该题目的验题人
3. 其他管理员不能再认领

**提交验题方案**：
1. 认领后出现「验题方案」输入框
2. 提交验证用的解决方案（Markdown）
3. 可多次更新

**取消认领**：
1. 点击「取消认领」
2. 验题方案不会丢失

## 可见性控制

需要 `manage_site` 权限。

题目发布后，可以通过「可见性设置」控制：

- 对特定用户可见
- 对特定权限组可见
- 对所有人可见（公开）
- 仅管理员可见

## 题目列表

- 公开列表仅显示 `approved` 和 `published` 状态的题目
- 管理员可通过 `?all=true` 查看全部状态的题目
- 支持按难度筛选
- 题目卡片显示：标题、作者、难度、状态

## 分配赛事

需要 `manage_contests` 权限。

1. 在题目详情页点击「分配赛事」
2. 选择目标赛事
3. 题目出现在该赛事的题目列表中
4. 可在赛事编辑中调整题目顺序
