# 🚀 新手上路

## 注册与登录

### 方式一：CP OAuth

如果你的站点配置了 CP（Codeforces 风格）OAuth：

1. 访问站点首页
2. 点击「登录」
3. 点击第三方 OAuth 登录按钮
4. 跳转到认证页面，授权后自动登录

### 方式二：演示模式（OAuth 不可用时）

1. 在登录页输入任意字符串作为 token
2. 系统会匹配预设的演示用户
3. 适用于本地开发测试

### 方式三：管理员密码登录

管理员账户（user_id=admin）可通过密码直接登录：

1. 访问登录页
2. 输入管理员密码
3. 点击登录

> 管理员密码在首次启动时通过 `ADMIN_PASSWORD` 环境变量或 `mcguffin init` 设置。

## 基本界面

| 区域 | 说明 |
|------|------|
| **顶部导航** | 站点名称、导航入口（题目、比赛、成员、社区、配置） |
| **右侧操作区** | 暗色模式切换、字体大小、通知铃铛、用户菜单 |
| **主体内容** | 根据导航动态切换 |

## 申请加入团队

1. 点击导航栏的「申请加入」
2. 填写申请表单（姓名、邮箱、申请理由）
3. 提交后状态变为「待审核」
4. 等待管理员审批
5. 通过后会获得团队成员权限

## 个人设置

1. 点击右上角头像 → 进入个人主页
2. 可查看和编辑：
   - 显示名称
   - 个人简介
   - 头像（通过外部链接）

## 通知系统

- 右上角铃铛图标显示未读通知数量
- 点击展开通知列表
- 可标记单条已读或全部已读
- 通知类型包括：审核结果、评论回复、@提及等
