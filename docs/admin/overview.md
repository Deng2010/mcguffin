# 👥 管理后台总览

管理后台是站点管理员的核心操作界面。访问 `/admin/config` 进入（需 `manage_site` 权限或 superadmin 角色）。

## 导航

左侧侧边栏包含以下标签页：

| 标签 | 路径 | 说明 |
|------|------|------|
| 配置 | `/admin/config` | 站点信息、难度等级、自动备份等运行时配置 |
| 用户管理 | `/admin/users` | 用户列表、角色变更、权限设置、权限组管理 |
| 角色权限 | `/admin/roles` | 权限概览与映射查看 |
| 插件管理 | `/admin/plugins` | WASM 插件安装/卸载/刷新 |
| 备份管理 | `/admin/backups` | 备份创建/恢复/下载、数据导入导出 |

底部面板可切换暗色模式 / 字体大小，返回主站。

## 权限要求

| 后台功能 | 所需权限 |
|----------|----------|
| 查看配置 | `manage_site` |
| 修改配置 | `manage_site` |
| 用户管理 | `manage_site` |
| 角色管理 | `manage_site` |
| 插件管理 | `manage_site` |
| 备份管理 | `manage_backups` |
| 重启服务 | `manage_site` |

## 基本操作

### 在线修改配置

1. 进入「配置」页面
2. 修改任意可编辑字段
3. 点击「保存」
4. 如配置项需重启，页面会提示，点击「重启服务」按钮生效

### 管理用户

1. 进入「用户管理」页面
2. 可查看所有用户列表（ID、用户名、角色、邮箱、注册时间）
3. 对每个用户可执行：变更角色、设置权限组、设置个人权限、移除用户

### 管理插件

1. 进入「插件管理」页面
2. 上传 `.wasm` 文件或输入 URL 安装插件
3. 对已安装插件可一键卸载
4. 点击「扫描目录刷新」同步文件系统的变化

### 数据备份

1. 进入「备份管理」页面
2. 点击「创建备份」手动备份
3. 可从列表中恢复、下载或删除备份
4. 可导出/导入数据和配置文件
