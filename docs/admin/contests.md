# 🏆 赛事管理

## 赛事状态流转

```
draft ──→ upcoming ──→ ongoing ──→ finished
  ↑                        │
  └────────────────────────┘
```

| 状态 | 说明 | 公开可见 |
|------|------|----------|
| `draft` | 草稿，仅管理员可见 | 否 |
| `upcoming` | 即将开始，公开显示但不可参赛 | 是（需 `view_public_contests`） |
| `ongoing` | 进行中 | 是 |
| `finished` | 已结束 | 是 |

## 操作指南

### 创建赛事

1. 进入「比赛」页面 → 点击「创建赛事」
2. 填写赛事信息：
   - 名称
   - 开始/结束时间
   - 描述（Markdown 格式）
   - 外部链接（可选）
3. 创建后默认为 `draft` 状态

### 编辑赛事

1. 进入赛事详情页
2. 点击「编辑」
3. 修改信息后保存

### 变更状态

1. 在赛事详情页点击「设置状态」
2. 选择目标状态
3. `draft → upcoming` 会公开赛事基本信息
4. `upcoming → ongoing` 开始计时
5. `ongoing → finished` 结束赛事

### 题目编排

1. 进入赛事详情页
2. 在「题目」区域：
   - 从已有题目中选择添加到赛事
   - 拖拽调整题目顺序
3. 保存后题目按顺序在赛事页面展示

### 可见性控制

可在赛事编辑中设置：

- **可见性 ACL**：限制哪些用户/组可以查看该赛事
- **可编辑 ACL**：限制哪些用户/组可以编辑该赛事

## 默认角色权限

| 操作 | 所需权限 |
|------|----------|
| 创建赛事 | `manage_contests` |
| 编辑赛事 | `manage_contests` |
| 删除赛事 | `manage_contests` |
| 变更状态 | `manage_contests` |
| 编排题目 | `manage_contests` |
| 查看草稿赛事 | `view_all_contests` |
| 查看公开赛事 | `view_public_contests` |
